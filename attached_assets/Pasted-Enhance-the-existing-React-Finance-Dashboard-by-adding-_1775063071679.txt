Enhance the existing React Finance Dashboard by adding an **AI-powered Financial Insights module** that analyzes user transactions and generates smart financial observations and predictions.

This feature must work **entirely on the frontend** without requiring any external AI API.

The goal is to simulate intelligent financial analysis using algorithms applied to transaction data.

---

FEATURE NAME

Smart AI Financial Insights

---

CREATE NEW COMPONENT

src/components/Insights/AIInsights.jsx

---

INSIGHTS TO GENERATE

Analyze transactions and generate insights such as:

1. Highest Spending Category

Determine which category has the highest total expense.

Example output:

"You spent the most on Food this month."

---

2. Monthly Expense Comparison

Compare the current month's expenses with the previous month.

Example output:

"Your expenses increased by 12% compared to last month."

or

"Great! Your expenses decreased by 8% compared to last month."

---

3. Spending Pattern Detection

Analyze if spending is concentrated in certain categories.

Example output:

"45% of your spending is on Food and Entertainment."

---

4. Savings Indicator

Calculate:

Savings = Income - Expenses

Example output:

"You saved ₹12,500 this month."

---

5. Next Month Expense Prediction

Predict next month expenses using a simple trend formula.

Prediction formula:

NextMonthPrediction =
Average(last 3 months expenses)
OR
CurrentMonthExpense * 1.05

Example output:

"Predicted expenses for next month: ₹18,200."

---

6. Financial Health Score

Generate a score from 0 to 100 based on:

Savings ratio
Income vs expenses
Spending distribution

Example output:

"Financial Health Score: 78/100 – Good financial balance."
