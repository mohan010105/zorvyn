Enhance the existing React Finance Dashboard application by implementing a **complete authentication flow** including **Login, Signup, and Reset Password pages**.

The authentication system should work fully on the frontend using **LocalStorage and React Context API** to simulate a real authentication system.

Do not break any existing dashboard features.

The application must still run correctly using:

npm install
npm run dev

AUTHENTICATION FLOW

1. SIGNUP PAGE

Create a signup page with fields:

Full Name
Email
Password
Confirm Password

Validation requirements:

Email must be valid format
Password minimum 6 characters
Passwords must match

When signup succeeds:

Save user data to LocalStorage
Automatically redirect to Login page

---

2. LOGIN PAGE

Create a login form with:

Email
Password

Login logic:

Check entered credentials against LocalStorage users
If valid:

Save logged-in user to LocalStorage
Redirect to Dashboard

If invalid:

Show error message.

---

3. RESET PASSWORD PAGE

Create a reset password form with:

Email
New Password
Confirm Password

Behavior:

If email exists in stored users
Update the password
Show success message
Redirect to Login page

---
