npm install
npm run dev

Use the following stack:

JavaScript (not TypeScript)

CORE FEATURES

1. DASHBOARD OVERVIEW

Display financial summary cards:

Total Balance
Total Income
Total Expenses

Include:

Balance trend chart (time based visualization)
Spending breakdown pie chart (categorical visualization)

Charts should use mock transaction data.

---

2. TRANSACTIONS SECTION

Create a transaction table with:

Date
Amount
Category
Type (income or expense)

Include features:

Search transactions
Filter by category
Filter by income/expense
Sort by amount or date

Transactions must be stored in global state and persisted to LocalStorage.

---

3. ROLE BASED UI

Simulate two roles:

Viewer
Admin

Use a dropdown toggle in the header to switch roles.

Viewer permissions:
Can only view transactions.

Admin permissions:
Can add transactions
Can edit transactions
Can delete transactions

Admin actions should open a modal form.

---

4. INSIGHTS SECTION

Automatically generate insights from the data:

Highest spending category
Monthly income vs expense comparison
Total savings calculation

Display insights as cards with icons and small charts.

---

5. STATE MANAGEMENT

Use React Context API to manage:

Transactions
Filters
User Role

Ensure components subscribe to context cleanly.

---

6. RESPONSIVE UI

Design a modern dashboard layout with:

Sidebar navigation
Header with role switch
Cards and charts grid

Ensure responsive behavior for:

Desktop
Tablet
Mobile

Use Tailwind CSS grid and flex utilities.
